## 运算符

### 运算符及其优先级

|优先级|操作符|描述                                   |
| ---- | --- | ------------------------------------ |
|1     |()   |调节优先级的括号操作符                  |
|      |[]   |数组下标访问操作符                      |
|      |->   |通过指向对象的指针访问成员的操作符        |
|      |.    |通过对象本身访问成员的操作符             |
|      |++   |后置自增操作符                          |
|      |--   |后置自减操作符                          |
|2     |!    |逻辑取反操作符                          |
|      |~    |按位取反操作符                          |
|      |++   |前置自增操作符                          |
|      |--   |前置自减操作符                          |
|      |-    |一元取负操作符                          |
|      |+    |一元取正操作符                          |
|      |*    |解引用操作符                            |
|      |&    |取地址操作符                            |
|3     |->*  |在指针上通过指向成员的指针访问成员的操作符 |
|      |.*   |在对象上通过指向成员的指针访问成员的操作符 |
|4     |*    |乘法操作符                              |
|      |/    |除法操作符                              |
|      |%    |取余操作符                              |
|5     |+    |加法操作符                              |
|      |-    |减法操作符                              |
|6     |<<   |按位左移操作符                          |
|      |>>   |按位右移操作符                          |
|7     |<    |小于比较操作符                          |
|      |>    |大于比较操作符                          |
|      |<=   |小于等于比较操作符                       |
|      |>=   |大于等于比较操作符                       |
|8     |==   |等于操作符                              |
|      |!=   |不等于操作符                            |
|9     |&    |按位与操作符                            |
|10    |^    |按位异或操作符                          |
|11    |\    |按位或操作符                            |
|12    |&&   |逻辑与操作符                            |
|13    |&vert;&vert;|逻辑或操作符                     |
|14    |?:   |三元条件操作符                          |
|15    |=    |赋值操作符                              |
|      |+=   |复合赋值操作符(加法)                     |
|      |-=   |复合赋值操作符(减法)                     |
|      |*=   |复合赋值操作符(乘法)                     |
|      |/=   |复合赋值操作符(除法)                     |
|      |%=   |复合赋值操作符(取余)                     |
|      |&=   |复合赋值操作符(按位与)                   |
|      |^=   |复合赋值操作符(按位异或)                 |
|      |&vert;=|复合赋值操作符(按位或)                 |
|      |<<=  |复合赋值操作符(按位左移)                 |
|      |>>=  |复合赋值操作符(按位右移)                 |
