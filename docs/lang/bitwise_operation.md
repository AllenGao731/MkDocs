## 位运算

### 位运算简介

**什么是位运算？**  
位运算，按二进制位进行的运算，逐位进行运算

**位运算的作用**  
快速、简洁，计算机最直接的运算方式，常用于优化算法等

### 运算规则
负数参与位运算时，用负数的补码参与运算，观察结果，若是负数，则要以该负数为补码求它的原码，原码是最终结果。  
> ~a = -(a+1)

### 位运算的应用/性质

**&**  
奇偶判断
```cpp
if(x&1) cout<<"奇数";
```
统计二进制中1的个数
```cpp
while(a){
    cnt++;
    a=a&(a-1);
}
```
二进制中最低位的1所表示的值
```cpp
int lowbit(int x){
    return x & (-x);
}
```

**|**
```cpp
a | b >= max(a,b);
```

**^**  
交换变量
```cpp
b = a ^ b;
a = a ^ b;
b = a ^ b;
```

**<<**
```cpp
a<<b = a*int(pow(2,b));
```

**>>**
```cpp
a>>b = a/int(pow(2,b));
```
